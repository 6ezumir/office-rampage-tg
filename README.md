# office-rampage-tg
# 🏢 OFFICE RAMPAGE - Telegram Mini App

Разрушай офис, снимай стресс и набирай очки!

## 🎮 ОСОБЕННОСТИ

- 📱 Оптимизировано для мобильных устройств
- 🤖 Интеграция с Telegram Mini Apps
- 👥 3 уникальных персонажа с разными способностями
- 🏢 Полноценная офисная локация
- 💥 Физика объектов
- 🏆 Таблица лидеров
- 📳 Haptic Feedback

## 👥 ПЕРСОНАЖИ

| Персонаж | Способность | Описание |
|----------|------------|----------|
| 👔 БОСС | 💪 Сильный удар | Наносит 50 урона, но медленный |
| 👨‍💻 ОФИСНЫЙ | ⚡ Быстрый | Высокая скорость передвижения |
| 🧹 УБОРЩИК | 💚 Восстановление | Кофе восстанавливает больше стресса |

## 🎯 КАК ИГРАТЬ

1. **Выбери персонажа** в главном меню
2. **Перемещайся** - тапни и тяни в любую сторону
3. **Бери предметы** - тапни по объекту
4. **Бросай предметы** - отпусти палец
5. **Ударь предмет** - двойной тап
6. **Специальные действия** - кнопки внизу экрана

## 🎛️ УПРАВЛЕНИЕ

- 👆 **Тап по объекту** - взять/бросить
- 👆👆 **Двойной тап по объекту** - ударить
- 👆 **Тап и тяни по пустому месту** - перемещение
- 👊 **Кнопка "УДАР"** - ударить ближайший объект
- ☕ **Кнопка "КОФЕ"** - снизить стресс (-20%)
- 😡 **Кнопка "ГНЕВ"** - увеличить стресс (+30%) и получить очки
- 🏠 **Кнопка "МЕНЮ"** - выход в главное меню

## 📊 СИСТЕМА

- **🏆 Очки** - набираются за разрушение предметов
- **😫 Стресс** - растёт со временем и от действий
- При стрессе 100% - игра заканчивается
- Уничтожь все предметы - победа!
